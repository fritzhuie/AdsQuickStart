Quick Start for Unity 5.2+

Other Guides: Native iOS | Native Android | Unity 4.6 to 5.1

Step 1 - Open your Unity Project

Your UDN Account is used to manage Unity Ads profiles.

Other members of your Organization have access to your Ads account as well.

Step 2 - Integrate Unity Ads Into Your Game

For versions of Unity older than 5.2, click here.

Enable Unity Ads in the Unity Services Panel (Window -> Services)

To open the Ads Services panel, click "Ads", and enable the SDK in your project.

Once Ads is enabled, use the following code to call an ad from any scene


  using UnityEngine;
  using UnityEngine.Advertisements;

  public class UnityAdsExample : MonoBehaviour
  {
    public void ShowRewardedAd()
    {
      if (Advertisement.IsReady("rewardedVideo"))
      {
        var options = new ShowOptions { resultCallback = HandleShowResult };
        Advertisement.Show("rewardedVideo", options); // Show an ad with the "rewardedVideo" placement
      }
    }

    private void HandleShowResult(ShowResult result) // Called when the video ad ends
    {
      if (result != ShowResult.Skipped)
      {
        // Player has watched an ad - Provide the reward
        // Give coins, lives, etc...
      }
    }
  }

That's All!

Your game should now display test ads when you call ShowRewardedAd(). Don't forget to disable Test Mode when your game goes live!

Additional examples and troubleshooting can be found here.

For additional questions, check out the forum or contact our Unity Ads Support.

Step 3 - Manage your profile in the Dashboard

Log into the Dashboard with your UDN Account, and find your game

Select your game, then choose a platform (iOS or Android)

From here, you can modify placements and other game-specific settings.

Incentivize your ads for more $$$$

It's important to inventivize users to watch ads because it improves user engagement, and substantially increases revenue.

A good implementations genernally include one or more of the following:

  • In-game currency reward
  • Extra lives at the start of the game
  • Point boosts for the next round

In the code example above, you'll see a method (HandleShowResult()) that gets called when an ad finishes playing.
